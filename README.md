<p><b>Json2Typescript</b> is a lightweight IntelliJ / WebStorm plugin that converts copied JSON data into TypeScript type definitions with one click.</p>

<p><b>Features:</b></p>
<ul>
  <li>Convert JSON to TypeScript types directly from the editor context menu</li>
  <li>Support nested objects and arrays</li>
  <li>Infer union types (e.g. <code>number | null</code>)</li>
  <li>Automatically mark optional fields (<code>?:</code>)</li>
  <li>Designed for HTTP / API response data</li>
</ul>

<p>No configuration required. Just copy JSON and generate types instantly.</p>
